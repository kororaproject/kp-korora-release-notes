1. Välkommen till Fedora 18

1.1. Välkommen till Fedora

Du kan hjälpa Fedoraprojektsamfundet att fortsätta förbättra Fedora om du skriver felrapporter och förbättringsförslag. Gå till Fel och funktionsönskemål på Fedoras wiki, för mer information om rapportering av fel och önskemål. Tack för din medverkan.
För att hitta mer generell information om Fedora, gå till följande webbplatser på Fedoras wiki (http://fedoraproject.org/wiki/):
  • Fedoraöversikt
  • Fedoras frågor och svar
  • Hjälp och diskussioner
  • Medverka i Fedoraprojektet

1.1.1. Behöver du hjälp?

Det finns ett antal platser där du kan få hjälp om du skulle stöta på problem.
Om du stöter på ett problem och skulle vilja ha hjälp, gå till http://ask.fedoraproject.org. Många svar finns redan där, men om du inte hittar ditt, kan du enkelt posta en ny fråga. Detta har fördelen att någon annan med samma problem också kan hitta svaret.
Du kan också få hjälp i kanalen #fedora på IRC-nätet irc.freenode.net. Tänk på att det på kanalen finns frivilliga som vill hjälpa till, men att folk som har kännedom om ett visst ämne kanske inte alltid finns tillgängliga.

1.2. Översikt

Som vanligt fortsätter Fedora att utveckla (RedHats bidrag) och integrera de senaste programmen med fri och öppen källkod. (Fedora 18 Features). Följande avsnitt ger en snabb överblick över de större ändringarna från den senaste utgåvan av Fedora.
För mer detaljer om funktionerna som finns med i Fedora 18, se individuella wikisidor med detaljer om funktionernas mål och utveckling: http://fedoraproject.org/wiki/Releases/18/FeatureList.

1.3. Återkoppling

Tack för att du tar dig tid att skicka dina kommentarer, förslag och\nfelrapporter till Fedorasamfundet. Detta hjälper till att förbättra kvaliteten på Fedora, Linux och fri programvara över hela världen.

1.3.1. Att ge återkoppling om Fedoraprogramvara

För att ge återkoppling om Fedoras programvara eller andra delar av systemet, gå till Bugs and Feature Requests. En lista på ofta rapporterade fel och kända problem med denna utgåva finns tillgänglig från Common F18 bugs, på wikin.

1.3.2. Att ge återkoppling om utgåvenoteringarna

Om du tycker att dessa utgåvenoteringar kan förbättras på något sätt, kan du ge återkoppling direkt till nyhetsskribenterna. Dett finns flera sätt att ge återkoppling, i ordning med de bästa först:
  • Om du har ett Fedorakonto, redigera innehållet direkt på Docs-Beats-sidan på wikin.
  • Fyll i en felrapport genom att använda denna mall - Denna länk är ENDAST för återkoppling om själva utgåvenoteringarna. Se råden ovan för detaljer.
  • Epost till sändlistan om utgåvenoteringar relnotes@fedoraproject.org

1.4. Katalogbyte för utgåvenoteringar

Utgåvenoteringarna kommer flyttas

Med början i denna utgåva har Fedoras utgåvenoteringar en ny plats.
Traditionellt har utgåvenoteringarna installerats i /usr/share/doc/HTML/fedora-release-notes/. Med tiden har den mesta dokumentationen som som fanns i katalogträdet /usr/share/doc/HTML/ flyttats bort därifrån till programspecifika kataloger.
För de flesta användare kommer denna ändring vara osynlig eftersom utgåvenoteringarna läses genom att välja menyingången. Några användare går dock direkt till filen, eller kan ha sina egna interna länkar.
Med början i Fedora 18 placeras utgåvenoteringarna i /usr/share/doc/fedora-release-notes-18.0/.
